2D Sonar Imaging System

The aim of this project is to design, build and test an imaging sonar operating at 40 kHz in air. This was achieved using the Teensy 3.6 micro-controller with three sonar transceiver. One of these transceivers acted as an emmiter, with the other two acting as receivers. Once a chirp signal was emitted, the system was able to processes any echo that had been recorded and made use of signal processing techniques to determine the distance and location of nearby objects.

This project was for a third year engineering design course and is meant to highlight the design capabilities of Electrical and Computer Engineering students. Furthermore, the project is meant to showcase our abilites in signal process and embedded systems design.

Unsupervised Image Cluster

This program makes use of the simple machine learning algorithm known as K-Means clustering to group any number of PPM images provided by the user. The application is primarily focused on grouping together images of digits, and would allow the user to sort the digits quickly and accurately without manual intervention. Functionality of features of this K-Means implementation include:

  • Ability to work on dataset of any size.
  • Ability to manipulate basic K-means parameters.
  • Ability run K-Means a specified number of times to find best clustering of data.
  • Ability to pick how inital centroids / clusters are initialised (K-Means++ included).
  • Ability to shuffle input data to make sure clustering is not affected by initial order of dataset.
  • Use HSV colour scheme to improve accuracy.

This project was for a third year computer science course and is intended to give users a basic introduction into machine learning with c++.
